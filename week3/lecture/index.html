CSS

Styling your HTML markup.

Can add borders, margins, colors, and more

Three ways to add styling:


  • Inside an element as an attribute (bad)
  • Inside a <style> tag in the html document (ehh)
  • Linking to an external CSS file (good)

Anatomy of a CSS Rule

Made of 3 main parts:

  • A Selector: the tag, class, or id you want to target
  • The Property of that tag you want to stylize
  • The Value of the attribute

Examples


p { color: red; }

h1, h2 {
	color: #132cff;
	font-weight: bold;
}

Classes and IDs

These are global attributes that can be applied to all html elements. They are used to identify particular tags, and can be used to attach styling.

  • IDs take priority over Classes
  • IDs are unique, and can only be used once per page
  • Classes can be reused as many times as needed
  • Both are case sensitive. Spelling counts.
  • Can be used on anything

Examples:



Dakota Ling



1337 Awesome Ave.


/* Multiple classes can be separated with spaces */

Atlanta, GA 30300

Tags can have multiple classes, but only one id. Classes are separated with spaces.

Targeting a Class

use a period . to target a class.


.col-2 { width: 200px }

img.highlight { border: 5px solid black }

footer .copy { font-size: 10px; }

Targeting an ID

Use the number symbol # (aka hashtag) to target an ID


#name { text-decoration: underline }

p#info { color: red; }

Psuedo Selectors

Used to select HTML tags under certain conditions

Common uses are for:

  • :hover states
  • :nth-of-type()
  • :nth-child()
  • :first-child
  • :last-child

The difference between :nth-child and :nth-of-type

Units

Different units of measure to set sizes

  • Pixels - 20px
  • Percent - 20%
  • Em - 1em
  • Rem - 1rem
  • And others, like pt, pc, cm, mm, in

We will be using px for now

Display Types

block:

  • Has a line break before and after it
  • Fills the width of its container

Display Types

inline:

  • Flows inline to its neighbor element
  • Only takes up as much width and height as its contents require

Display Types

inline-block:

  • The inside acts as a block level element, while the outside acts as inline.

Display Types

none:

  • The element is not displayed, duh
  • Does not affect document flow
